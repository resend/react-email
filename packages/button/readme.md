## Installation

#### With yarn

```sh
yarn add -E @react-email/button
```

#### With NPM

```sh
npm install --save-exact @react-email/button
```

## Getting Started

Add the `Button` to your app first.

```jsx
import { Button } from '@react-email/button';

const App = () => {
  return (
    <div>
      <Button href="https://react.email">Button</Button>
    </div>
  );
};
```
