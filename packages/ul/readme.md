## Install

Install component from your command line.

#### With yarn

```sh
yarn add @react-email/ul -E
```

#### With npm

```sh
npm install @react-email/ul -E
```

## Getting started

Add the component to your email template. Include styles where needed.

```jsx
import { Ul } from '@react-email/ul';

const Email = () => {
  return (
    <Ul>
      <li>foo</li>
      <li>bar</li>
    </Ul>
  );
};
```
