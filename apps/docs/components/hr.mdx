## Install

Install component from your command line.

<CodeGroup>

```sh npm
npm install @react-email/hr -E
```

```sh yarn
yarn add @react-email/hr -E
```

</CodeGroup>

## Getting started

Add the component to your email template. Include styles where needed.

```jsx
import { Hr } from '@react-email/hr';

const Email = () => {
  return <Hr />;
};
```
