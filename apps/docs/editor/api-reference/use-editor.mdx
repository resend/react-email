The React Email package previously exported a `useEditor` helper from `@react-email/editor/core`. That export has been removed. Use TipTap’s [`useEditor`](https://tiptap.dev/docs/editor/api/react-hooks/use-editor) from `@tiptap/react` instead, together with extensions from `@react-email/editor/extensions` (for example `StarterKit`).

```tsx
import { StarterKit } from '@react-email/editor/extensions';
import { useEditor, EditorContent } from '@tiptap/react';

export function MyEditor() {
  const editor = useEditor({
    extensions: [StarterKit],
    content: '<p>Start typing.</p>',
  });

  if (!editor) {
    return null;
  }

  return <EditorContent editor={editor} />;
}
```

See [Getting started](/editor/getting-started) for a full setup with `EditorProvider` and UI components.
