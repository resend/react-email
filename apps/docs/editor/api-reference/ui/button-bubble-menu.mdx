Contextual menu that appears when clicking an email button.

## ButtonBubbleMenu.Default

Pre-configured button bubble menu with edit link and unlink actions.

```tsx
import { ButtonBubbleMenu } from '@react-email/editor/ui';

<ButtonBubbleMenu.Default />
```

<ResponseField name="placement" type="'top' | 'bottom'" default="'bottom'">
  Position relative to the button.
</ResponseField>

<ResponseField name="offset" type="number" default="8">
  Distance from the button in pixels.
</ResponseField>

<ResponseField name="onHide" type="() => void">
  Called when the bubble menu is hidden.
</ResponseField>

<ResponseField name="className" type="string">
  Additional CSS class on the wrapper element.
</ResponseField>

## Compound components

For custom button bubble menus, compose with these sub-components:

| Component | Description |
|-----------|-------------|
| `ButtonBubbleMenu.Root` | Base container — sets up context and positioning |
| `ButtonBubbleMenu.Toolbar` | Wrapper that hides when editing mode is active |
| `ButtonBubbleMenu.EditLink` | Button that enters editing mode to change the link URL |

### Example

Here's how `ButtonBubbleMenu.Default` is composed internally — use this as a
starting point for your own custom button bubble menu:

```tsx
import { ButtonBubbleMenu } from '@react-email/editor/ui';

<ButtonBubbleMenu.Root placement="top" offset={8}>
  <ButtonBubbleMenu.Toolbar>
    <ButtonBubbleMenu.EditLink />
  </ButtonBubbleMenu.Toolbar>
</ButtonBubbleMenu.Root>
```

The `Toolbar` component automatically hides when the user clicks `EditLink` and
enters editing mode. When editing is finished, the toolbar reappears.

## Context

Use `useButtonBubbleMenuContext()` inside custom children of `ButtonBubbleMenu.Root`
to access the editor and editing state:

```tsx
import { useButtonBubbleMenuContext } from '@react-email/editor/ui';

function CustomToolbarItem() {
  const { editor, isEditing, setIsEditing } = useButtonBubbleMenuContext();

  return (
    <button onClick={() => setIsEditing(true)}>
      Edit URL
    </button>
  );
}
```

| Field | Type | Description |
|-------|------|-------------|
| `editor` | `Editor` | The TipTap editor instance |
| `isEditing` | `boolean` | Whether the menu is in editing mode |
| `setIsEditing` | `(value: boolean) => void` | Toggle editing mode |
